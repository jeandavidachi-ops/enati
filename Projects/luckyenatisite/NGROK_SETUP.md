# Настройка ngrok для вашего сайта

## Что такое ngrok?
ngrok создает безопасный туннель к вашему локальному серверу, позволяя получить публичный URL для доступа к вашему сайту из интернета.

## Пошаговая инструкция

### 1. Установка ngrok
Если ngrok не установлен, скачайте его с https://ngrok.com/download

### 2. Запуск серверов

#### Вариант A: Использование batch файлов (рекомендуется)
1. Запустите `start-server.bat` - это запустит Node.js сервер
2. В новом окне терминала запустите `start-ngrok.bat` - это создаст туннель

#### Вариант B: Ручной запуск
1. **Запустите Python сервер** (если у вас есть):
   ```bash
   python your_python_server.py
   ```

2. **Запустите Node.js сервер**:
   ```bash
   npm start
   ```

3. **Запустите ngrok**:
   ```bash
   ngrok http 3001
   ```

### 3. Получение публичного URL
После запуска ngrok вы увидите что-то вроде:
```
Session Status                online
Account                       your-email@example.com
Version                       3.x.x
Region                        United States (us)
Latency                       45ms
Web Interface                 http://127.0.0.1:4040
Forwarding                    https://abc123.ngrok.io -> http://localhost:3001
```

**Ваш публичный URL будет**: `https://abc123.ngrok.io`

### 4. Проверка работы
1. Откройте ваш публичный URL в браузере
2. Убедитесь, что сайт загружается
3. Проверьте, что API запросы работают (например, загрузка данных в блоках)

## Важные моменты

### Безопасность
- ngrok URL меняется при каждом перезапуске (в бесплатной версии)
- Для постоянного URL нужна платная подписка
- Не передавайте ngrok URL третьим лицам без необходимости

### Ограничения бесплатной версии
- URL меняется при каждом перезапуске
- Ограничение на количество одновременных туннелей
- Ограничение на трафик

### Мониторинг
- Web интерфейс ngrok доступен по адресу: http://127.0.0.1:4040
- Здесь вы можете видеть все HTTP запросы и ответы

## API Endpoints
Ваш сервер предоставляет следующие API:
- `/api/top-groups` - топ группы
- `/api/top-coins/:groupId` - топ монеты группы
- `/api/shared-contracts` - общие контракты (TOP CLANS)
- `/api/latest-records` - последние записи (LATEST COINS)
- `/api/token-image/:contractAddress` - изображения токенов
- `/api/all-groups-stats` - статистика всех групп

## Устранение проблем

### Сервер не запускается
- Проверьте, что порт 3001 свободен
- Убедитесь, что установлены все зависимости: `npm install`

### ngrok не работает
- Проверьте, что ngrok установлен: `ngrok version`
- Убедитесь, что Node.js сервер запущен на порту 3001

### API запросы не работают
- Убедитесь, что Python сервер запущен на порту 5000
- Проверьте логи в консоли браузера (F12)

## Полезные команды
```bash
# Проверить, что сервер запущен
netstat -an | findstr :3001

# Проверить версию ngrok
ngrok version

# Запустить ngrok с кастомным доменом (требует платную подписку)
ngrok http 3001 --domain=your-custom-domain.ngrok.io
```
